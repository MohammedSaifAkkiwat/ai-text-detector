About the AI Text Detector

Project Overview

This web application is a tool designed to identify whether a given piece of text is likely to have been generated by an artificial intelligence model or written by a human. It leverages a machine learning classifier trained on a diverse dataset of both human and AI-authored content.

The goal of this project is to explore the capabilities of machine learning in natural language processing and to provide a practical demonstration of text classification techniques.

How It Works

Users can input text directly into the provided text area or upload a .txt file. Upon submission, the application processes the text using a trained machine learning model (Random Forest). The model analyzes the linguistic features of the text and predicts whether it was likely generated by an AI or a human. The prediction is then displayed to the user, along with the original text.

A basic feedback mechanism is also included, allowing users to indicate whether the model's prediction was correct. While this feedback is not currently stored persistently, it demonstrates the potential for future model improvement through user interaction.
